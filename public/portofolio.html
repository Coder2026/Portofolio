Healthy

As part of the Human-Computer Interaction (HCI) course, I designed the user interface (UI) for the Healthy app, focusing on creating an intuitive user experience. This mobile application concept is aimed at promoting a healthy lifestyle by monitoring physical activities through smartwatch integration. It also offers personalized recommendations for nutritious food and helps users locate gyms and workout routines that suit their needs. Additionally, users can explore a wide range of health-related articles.

I designed three key sections of the app:

- Login, Shop, and Schedule: Each section was designed with user-friendly interfaces and seamless navigation to ensure smooth user experiences. My focus was on crafting a straightforward yet effective UI to enhance usability.

On Time

As part of my Software Engineering class, I developed the Ontime mobile application, taking on multiple roles as Frontend Developer, Backend Developer, and Scrum Master. Ontime is a task management app designed to help users effectively manage their tasks and schedules.

Key responsibilities:

- Login, Register, and Reset Password: I developed both the UI (using XML for layout) and the backend logic (using Java) to handle user authentication and password recovery.
- Todo and Schedule Management: I implemented a Todo feature allowing users to manage tasks, and I also designed the Schedule feature for organizing daily or weekly plans.

This project demonstrates my ability to manage both frontend and backend development, while also applying agile principles in my role as Scrum Master.

Cocokien

As part of my Entrepreneur Course, I developed the backend for Coocokien, a website designed to market products with an innovative code redemption system. The highlight of the project is a gacha mechanism where users redeem codes to receive virtual stickers, which can later be used for promotional purposes or exchanged for rewards.

Key technologies and features:

- Code Redemption & Gacha System: I developed the backend to manage code redemptions and integrated the gacha system to distribute stickers randomly.
- OCR Integration: I implemented Optical Character Recognition (OCR) to scan and automatically read codes from images, simplifying user interactions.
- Backend Development: Using Flask and Python, I built a robust backend that handles code processing, gacha logic, and system integration.
